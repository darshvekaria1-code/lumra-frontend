# Logo Files

Place your IB and IGCSE logo images in this directory:

- `ib-logo.png` - International Baccalaureate logo (recommended size: 32x32px or 64x64px, transparent background preferred)
- `igcse-logo.png` - Cambridge IGCSE logo (recommended size: 32x32px or 64x64px, transparent background preferred)

## Supported Formats
- PNG (recommended for transparency)
- SVG (scalable, best quality)
- JPG/JPEG (if no transparency needed)

## Notes
- The logos will automatically scale to fit (32x32px display size)
- If images are not found, the code will fallback to text badges
- Make sure the files are named exactly as shown above
